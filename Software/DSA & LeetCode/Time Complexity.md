# Time Complexity

> A blog is just like having a long conversation with people, so it should make sense that things you enjoy talking about will be closely related to your passion.

![[Pasted image 20251125154607.png]]

|Complexity|Name|When it Happens|
|---|---|---|
|**O(1)**|Constant|Direct access, array index, hash lookup|
|**O(log n)**|Logarithmic|Binary search|
|**O(n)**|Linear|Loop over all elements|
|**O(n log n)**|Linearithmic|Merge sort, quicksort average|
|**O(n²)**|Quadratic|Nested loops|
|**O(2ⁿ)**|Exponential|Subset / backtracking brute force|
|**O(n!)**|Factorial|Permutations|
