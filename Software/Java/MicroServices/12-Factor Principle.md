# 12-Factor Principle

> A blog explains 12-Factor Principles of Cloud-Native Microservices


## 12-Factor

- Codebase
- Config
- Release & Run
- Port Binding
- Disposability
- Logs
- Dependencies
- Backing services
- Processes
- Concurrency
- Dev/Prod parity
- Admin processes
